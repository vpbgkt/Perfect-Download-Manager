Changelog

Release notes for Perfect Download Manager (PDM). PDM updates itself automatically — existing installs are offered each new stable release from inside the app (toolbar menu → Check for Updates). You can also download the latest version any time.

1.0.26

Latest stable

  • Much faster resume. Paused downloads now re-spread the remaining data across all connections when resumed, so a resumed transfer runs at full speed instead of crawling on one connection.
  • Refreshed links on resume. When a download is resumed, PDM re-resolves the source link, so expiring CDN and file-host links keep working.
  • Smarter duplicate detection. Starting a download PDM already has now recognises the same file even when the link changes each time (Google Drive, signed links), and offers to continue the existing partial download, open the finished file, or start a fresh copy.
  • Change or refresh a download's link. Right-click a download to paste a new link, or re-capture a fresh one from your browser, and continue where you left off.
  • Smaller, self-setup installer. The new installer is about 41 MB and installs the .NET 10 runtime automatically if it's missing — no separate download.
  • Support & contact. Reach the team for help or license renewals from the toolbar menu and the support page.
  • Lower memory usage while idle, and a refreshed green installer.

1.0.18

  • Reliability and UX improvements to the download engine and browser capture.
  • Signed, verified auto-updates (ECDSA + SHA-256).
